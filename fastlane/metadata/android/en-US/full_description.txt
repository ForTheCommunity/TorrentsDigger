<p>
<b> Torrents Digger </b> digs Torrents from multiple sources/sites.
</p>

<p>
<b> --->> Features :~ </b>
</p>

<ul>
    <li> -> Multi Source/Sites Support. </li>
    <li> -> Shows Torrent Creation Date, Seeders, Leechers, Total Downloads, Torrent Size. </li>
    <li> -> User can choose Category, Filters, Sortings, Sortings Order. </li>
    <li> -> Bookmark : User can Bookmark torrents. </li>
    <li> -> Copy magnet link to Clipboard. </li>
    <li> -> Opens Torrent Downloader Apps for Downloading Torrents. </li>
    <li> -> Default Trackers Lists : User can Select a Default Trackers List & that trackers List will be added to magnet links. </li>
    <li> -> Custom Listings : There are numbers of Custom Listings that shows certain types of torrents. (Like Trending , Latests, etc.) </li>
    <li> -> Multiple Themes. </li>
    <li> -> Proxy Support : User can use Proxy to route Traffic. Supported Proxy Protocols -> ( HTTP, HTTPS, SOCKS4 & SOCKS5. ) </li>
    <li> -> Database Import/Export : User can Export and Import App Database. </li>
    <li> -> Cross Platform ( Currently Available for Android & Linux ). </li>
</ul>
